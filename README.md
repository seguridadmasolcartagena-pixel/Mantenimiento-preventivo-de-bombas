# Mantenimiento-preventivo-de-bombas
En esta aplicación se va poder consultar el historial de las mediciones de vibración de las bombas de la planta, así como sus incidencias y gráficas, con el objetivo de anticipar la rotura de una bomba.
Para introducir nuevos datos hay que seleccionar "importar excel" y subir el excel previamente descargado del equipo de medición.
